Maquetación tradicional con CSS: Centrado del layout

Cuando hablamos de centrar un elemento o etiqueta, siempre lo hacemos con respecto a su elemento o etiqueta padre. Tendremos dos tipos de centrado: vertical y horizontal.

Centrado horizontal

  • Elementos en línea: text-align:center; (al padre)
  • Elementos en bloque:
    • margin: [X] auto; (al elemento. Debe de tener anchura)
  • Varios elementos en Bloque en la misma fila:
    • text-align: center; (al padre)
    • display: inline-block (a los elementos)
    • Deben de tener anchura
  • Con contenedores flex (próximos cursos)

Centrado vertical

Elementos en línea

  • El mismo padding arriba y abajo
  • vertical-align:middle si está dentro del elemento de tabla o lo estamos simulandolo con la propiedad display. (El padre debe tener altura fija)
  • Con contenedores flex (próximos cursos)
