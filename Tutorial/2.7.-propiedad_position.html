Maquetación tradicional con CSS: Propiedad position

La propiedad position me va a permitir romper el flujo normal de los elementos. Se utiliza para detalles más finos en la maquetación.

Valores de la propiedad position

  • static (propiedad por defecto)
  • relative
  • absolute
  • fixed
  • sticky
  • inherit

Todos estos valores van unidos a las propiedades: top, bottom, left, right (desplazamiento) y z-index (capas, para que cuando haya objetos que se solapen, pueda indicar cual quiero que se vea primero)

Explición de cada valor de la propiedad position

static

Es el valor por defecto

El elemento sigue el flujo que le corresponde. Aunque se use las propiedades unidad de desplazamiento y z-index, NO se aplican.

relative

Como static pero SÍ atiende a las propiedades unidad de posiciónamiento o z-index. Lo que va a hacer es desplazarse hacia donde indiquemos desde la posición de flujo por defecto.

fixed

Establezco una posición a la caja con las propiedades de posicionamiento y z-index con respecto a la pantalla de mi web, y ya no se va a mover de ahí.

absolute

Se comporta igual que fixed pero se posiciona la caja tomando como referencia la primera etiqueta antecesora que posea el valor de position igual a relative. Es para posicionar en cualquiera posición, dentro de un contenedor, a un hijo.

sticky

Relative hasta llegar a una posición del scroll que lo hace convertirse en fixed.

inherit

La propidad position no actua en cascada, si queremos que los hijos se vean afectados por el valor position del padre, deberemos añadir a estos hijos el valor inherit.

Z-INDEX

Z-INDEX sirve para que, si dos o más elementos se solapan, decir cual va por encima del otro. Esta propiedad me permite añadir de esta manera capas a mi HTML. El z-index más alto va delante de todos los menores a él.

Centrado vertical con position (para elementos de bloque)

Si conocemos la altura del elemento y esta es or ejemplo 150px;

  • .contenedor{
    • position: relative;
  • }
  • .elemento_a_centrar{
    • height: 150px;
    • margin-top: -75px; /**la mitad de la altura**/
    • position: absolute;
    • top: 50%;
  • }

Si no conocemos la altura

  • .contenedor{
    • position: relative;
  • }
  • .elemento_a_centrar{
    • position: absolute;
    • top: 50%;
    • transform: translateY(-50%);
  • }
