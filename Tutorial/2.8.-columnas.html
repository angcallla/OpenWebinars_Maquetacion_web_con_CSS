Maquetación tradicional con CSS: Columnas

En CSS, podemos dividir el contenido de una caja en varias columnas como podemos ver en un periódico

Esto se hace con una serie de propiedades CSS

Propiedades CSS

  • column-count: Donde le digo al contenedor padre cuantas columnas quiero que tenga
  • column-width: Para fijar el ancho de las columnas. Si no se pone, el navegador es quien lo calcula. Atendiendo al ancho que pongamos, el navegador determinará cuantas columnas vamos a tener.
  • column-gap: Separación entre columnas
  • column-rule: Si quiero mostrar la línea que separa las columnas y con qué estilo
  • column-span: Si quiero que algun elemento salga de la separación entre columnas y ocupe todas ellas valores (all y none)
  • column-fill: Si quiero que las columnas se igualen en altura. Para ello, el padre deberá tener una altura determinada. Valores auto o balance (todas las columnas la misma altura)
  • break-inside:avoid: Si, por ejemplo, no quiero que un texto se parta de una columna a otra.
ESTAS SON PROPIEDADES NUEVAS EN CSS3 Y NO TODOS LOS NAVEGADORES LAS SOPORTAN
